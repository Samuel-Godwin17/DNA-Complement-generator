# 🧬 DNA Complement Generator

A simple Python project that takes a DNA base as input and returns its complementary base using basic `if / else` conditions.

---

## 📌 Project Description

In DNA, each base pairs with another specific base:

* Adenine (A) ↔ Thymine (T)
* Cytosine (C) ↔ Guanine (G)

This program asks the user to input a DNA base and prints its complementary base.

---

## 🚀 Features

* Beginner-friendly Python project
* Uses only `if / else` statements
* Handles invalid inputs
* Great for learning basic bioinformatics logic

---

 How It Works

The program checks the input using `if / else` conditions:

* If input is `A` → output `T`
* If input is `T` → output `A`
* If input is `C` → output `G`
* If input is `G` → output `C`
* Otherwise → invalid input
